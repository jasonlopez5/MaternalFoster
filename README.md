# MaternalFoster
MaternalFoster is a SwiftUI-based iOS application designed to provide accessible and reliable maternal health resources throughout pregnancy and post-birth care. The app integrates trusted public health resources from the CDC and MyHealthfinder APIs while offering tools such as week-by-week pregnancy tracking, due date calculation, health guidance, and post-birth care.

The goal of the project is to make important pregnancy-related information more accessible through a clean, easy-to-use mobile experience.
## Features
* Google Sign-In authentication using Firebase Authentication
* Week-by-week pregnancy tracking
* Due date calculator
* Health and nutrition resources
* Warning signs and maternal health guidance
* Post-birth care resources
* Dynamic sidebar navigation
* Firebase Firestore integration
* CDC and MyHealthfinder API integration

## Technologies Used
* SwiftUI
* Firebase Authentication
* Firebase Firestore
* Google Sign-In
* CDC Content Services API
* MyHealthfinder API
